# Academic Website - Quick Reference

## 🚀 Quick Start
Run `hugo server` to start.

## 📁 Content
- **Profile**: `content/authors/admin/_index.md`
- **Publications**: `content/publications/` (Create folders per paper)
- **CV**: Replace `static/uploads/resume.pdf`

## 🛠 Troubleshooting
If you see layout errors, run:
`hugo mod clean`
`hugo mod tidy`
